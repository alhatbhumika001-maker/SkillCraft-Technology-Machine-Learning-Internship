# SkillCraft Technology Machine Learning Internship

This repository contains the tasks completed during my Machine Learning Internship at SkillCraft Technology.

## Tasks

### Task 1 – House Price Prediction
Implemented Linear Regression to predict house prices based on features such as square feet, bedrooms, and bathrooms.

### Task 2 – Customer Segmentation
Used K-Means Clustering to group retail customers based on purchasing behavior.

### Task 3 – Cats vs Dogs Image Classification
Implemented Support Vector Machine (SVM) for classifying images of cats and dogs.

### Task 4 – Hand Gesture Recognition
Developed a real-time hand gesture recognition system using OpenCV and MediaPipe.

## Technologies Used

- OpenCV

- Scikit-learn
- NumPy
